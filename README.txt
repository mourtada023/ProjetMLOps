# Projet MLOps – Analyse prédictive du salaire

## Objectif
Prédire la probabilité d'obtenir un salaire élevé (>50K) à partir de caractéristiques socio-démographiques.

## Données
Source : Données revenus.csv (données du cours Scoring - extraction anonymisée).

## Méthodologie
- Exploration des données
- Prétraitement
- Entrainement
- Modélisation (Logistic Regression, Random Forest)
- Évaluation
- Automation

## Structure du projet
- data : raw, processed, modelisation 
- notebooks : exploration, preprocessing, modelisation, automatisation
- output : figures (graphs, tableaux, métrics score, matrices), jobs (modele final enregistré : Random Forest)
- rapport
- README

## Exécution
Les notebooks peuvent être exécutés dans l'ordre 01 → 04.
