# Projet MLOps – Analyse prédictive du salaire

## Objectif
Prédire la probabilité d'obtenir un salaire élevé (>50K) à partir de caractéristiques socio-démographiques.

## Données
Source : Données revenus.csv (données du curs Scoring - extraction anonymisée).

## Méthodologie
- Exploration des données
- Prétraitement
- Feature engineering
- Modélisation (Logistic Regression, Random Forest)
- Évaluation
- Automatisation

## Structure du projet
- data : raw, processed, modelisation 
- notebooks : exploration, preprocessing, modelisation, automatisation
- (src : load_data, preprocess, features, train_model, evaluate, automatisation)
- output : metrics, prediction, figures (graphs, tableaux, métrics score), docs(description datasat, variables, cible, methodologie)
- rapport
- README
- requierements

## Exécution
Les notebooks peuvent être exécutés dans l'ordre 01 → 04.
